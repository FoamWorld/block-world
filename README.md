## 关于
### 操作
* 空格 暂停/取消暂停
* `w`, `a`, `s`, `d` 上左下右移动
* `j`, `k` 靠近/远离鼠标位置
* `e` 使用方块
* `f` 使用物品
* `q` 丢弃
* `i` 显示信息
